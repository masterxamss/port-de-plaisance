The purpose of this tutorial is to show you how to create new accounts and perform operations such as adding, reading, editing, deleting


Create a new User

1 : To create a new user, click on the Users button on the Dashboard menu

2 : The list of all users will be displayed. To add a new user, click on the Add button

3 : Next, fill in the form and click Save

4 : Once the user has been successfully created, you will be redirected to the list of users.

List of Users

1 : To access the list of all users, in the left side menu of the dashboard select the Users option

Edit User

1 : To edit a user, click on the edit yellow button in the column options of the users list

2 : After that, a form with the user data will be displayed. Edit the data and click Save

Delete User

1 : To delete a user, click on the delete red button in the column options of the users list, confirm the intended action
