Tutorial: Reservations

Reservations

Reservatins

The purpose of this tutorial is to show you how to handle reservations and perform operations such as adding, reading, deleting


Create a new reservation

1 : Reservations are made according to a catway. To make a reservation, click on the Book now button on the card of the desired catway.

2 : Fill in the form and click Add

3 : If you try to make a reservation on a date that is already booked, an error message will be displayed.

4 : Once the booking has been successful, you will be redirected to the list of catways, and you will see that the color of the catway in question has changed, indicating that there is an active booking.

List of reservations

1 : To access the list of all reservations, in the left side menu of the dashboard select the reservations option

2 : The list of reservations will be displayed. To select a particular reservation, click on the green table icon in the options column.

3 : The details of the reservation will be displayed

4 : To list all reservations for a particular catway, select the reservations option in the catway details, then all reservations for that catway will be listed.

Delete reservation

1 : To delete a reservation, click on the delete button in the sidebar of the menu and then confirm.
