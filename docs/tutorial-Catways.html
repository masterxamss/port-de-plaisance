Tutorial: Catways

Catways

Catways

The purpose of this tutorial is to show you how to handle catways and perform operations such as adding, editing, updating, deleting


Create a new catway

1 : In the dashboard, select “Catways” from the left-hand menu.

2 : The list of catways will be rendered, each card representing a catway. Select the icon + in the top left corner

3 : Fill in the form and click Add

4 : If an error occurs, a message will be displayed in the bottom right corner

5 : After successfully adding the catway you will be redirected to the list of catways

Edit an existing catway

1 : Select Details on the card of the desired catway.

2 : The details of the catway will be displayed, then select Edit in the sidebar menu.

3 : A form with the catway data will be displayed.

4 : In the event of an error, a message will be displayed

5 : If successful, a message will be displayed

Update the state of a catway

1 : On details on the catway select Edit state.

2 : After edit a message will be displayed.

Delete catway

1 : To delete a catway, select delete in the left-hand side menu in the catway details.

2 : Then confirm your intention to delete the catway.

3 : A catway with active reservations cannot be removed. An error message will be displayed.

4 : After successful deletion you will be redirected to the list of catways. A success message will be displayed.
